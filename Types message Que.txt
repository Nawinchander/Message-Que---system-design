Types of Message Queues (Important for Interviews)
1. Point-to-Point (Queue model)
One producer → One consumer
Message consumed once

👉 Example: Order processing

2. Publish-Subscribe (Pub/Sub)
One producer → Multiple consumers

👉 Example: Notification system

Email service
SMS service
Analytics

3. Priority Queue
Messages processed based on priority

👉 Example: VIP users

4. Dead Letter Queue (DLQ)
Failed messages stored separately

👉 Important FAANG concept

⚙️ Popular MQ Tools (Mention in Interviews)
RabbitMQ
Apache Kafka
Amazon SQS
Redis
